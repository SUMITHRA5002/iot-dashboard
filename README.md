# IoT Dashboard - Water Monitoring System

## Description
This project is an IoT-based water monitoring system with leakage detection. It monitors water levels and detects leakage in real-time.

## Features
- Real-time data monitoring
- Leakage detection
- Dashboard visualization
- Backend + Frontend integration

## Technologies Used
- Frontend: HTML, CSS, JavaScript (React)
- Backend: Node.js
- Hardware: ESP board

## How to Run
1. Run backend server
2. Start frontend
3. Connect IoT device
